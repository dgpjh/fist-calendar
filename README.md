# 拳头记月法 - 月份天数记忆工具

一个有趣的交互式网页，通过拳头记忆法帮助学习每个月的天数，配有可爱的童声播报。

## ✨ 特性

- 🤛 **拳头记忆法可视化**：通过左手拳头的凸起和凹陷记忆大小月
- 🎤 **真实童声播报**：使用 DubbingX 智声云配 API 生成的专业童声（5岁女孩"圆圆"）
- 🎨 **Windows 98/7 怀旧风格**：经典的窗口界面设计
- ⚡ **快速轮播**：1.5倍速播报 + 7ms 间隔
- 🎉 **庆祝动效**：最终展示带有金色光晕呼吸效果

## 🎬 演示

1. 选择一个月份（1-12月）
2. 系统从一月开始依次播报到选定月份
3. 轮播结束后，显示大样式最终结果
4. 边框闪烁金色庆祝光效

## 🚀 使用方法

1. 克隆仓库：
```bash
git clone https://github.com/dgpjh/fist-calendar.git
cd fist-calendar
```

2. 用浏览器打开 `fist-calendar.html`

## 📁 文件结构

```
.
├── fist-calendar.html     # 主页面文件
├── fist.jpg               # 拳头图片
└── month-audio/           # 童声音频文件
    ├── 01.mp3  (一月大)
```

## 🎤 音色信息

- **音色**：圆圆（5岁女孩）
- **特点**：甜美、清脆、稚嫩
- **来源**：DubbingX 智声云配平台
- **语速**：1.5倍（活泼快速）

## 🎨 设计风格

- **UI 风格**：Windows 98/7 经典界面
- **配色**：米黄色背景 + 蓝色标题栏
- **按钮**：3D 凸起效果
- **边框**：立体浮雕样式

## 🛠️ 技术栈

- 纯 HTML + CSS + JavaScript
- Web Audio API（音频播放控制）
- DubbingX TTS API（童声生成）

## 📝 记忆口诀

> 一三五七八十腊（12月），三十一天永不差  
> 四六九冬（11月）三十日，平年二月二十八

或通过拳头：
- **凸起的指关节** = 大月（31天）
- **凹陷的间隙** = 小月（30天）
- **二月特殊** = 平年28天，闰年29天
